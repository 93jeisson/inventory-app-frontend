# 🎨 Inventory App Frontend

Frontend de un sistema de inventario desarrollado con React.

Este proyecto consume una API REST desarrollada con Spring Boot para la gestión de productos, usuarios y autenticación con JWT.

---

## 🚀 Tecnologías

* React
* JavaScript
* Axios o Fetch
* React Router
* Bootstrap / Tailwind (si usas alguno)

---

## 🔗 Backend del proyecto

Este frontend consume la siguiente API:

👉 https://github.com/93jeisson/inventory-app-api

---

## ⚙️ Instalación

```bash id="f1"
git clone https://github.com/93jeisson/inventory-app-frontend.git
cd inventory-app-frontend
npm install
npm start
```

---

## 🌐 Ejecución

La aplicación corre en:

```
http://localhost:3000
```

---

## 🔌 Conexión con backend

El frontend consume la API en:

```
http://localhost:8080/api
```
